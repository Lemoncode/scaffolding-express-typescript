## Passos

- Ejecutar en consola install

```bash
npm install
```

- Crear el archivo `.env` con el mismo contenido de `.env.example`.

- Ejecutar en modo de desarrollo (development):

```bash
npm start
```

- Ejecutar en modo debug:

```bash
npm run start:debug
```

- Y ejecutar `.vscode/launch.json`
